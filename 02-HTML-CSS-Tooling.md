#### Emmett Abbreviations
[A complete cheatsheet](https://docs.emmet.io/cheat-sheet/)

*Element:* `div` => 
```<div></div>```

*Child:* `ul>li` => 
```
<ul>
  <li></li>
</ul>
```

*Sibling:* `div+p+bq` => 
```
<div></div>
<p></p>
<blockquote></blockquote>
```

*Multiplication:* `ul>li*5` =>
```
<ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>
```

*Adding an Id or Class:* `ul#my-list>li.list-item*5` =>
```
<ul id="my-list">
    <li class="list-item"></li>
    <li class="list-item"></li>
    <li class="list-item"></li>
    <li class="list-item"></li>
    <li class="list-item"></li>
</ul>
```
